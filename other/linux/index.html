创建桌面图标

文件名: idea.desktop

[Desktop Entry]
Type=Application
Terminal=false 
Name=Navicat//应用名
Categories=Development;  // 应用分类
Icon=/opt/work/ide/navicat/navicat.png  //应用图标的绝对路径
Exec=/opt/work/ide/navicat/start_navicat  //应用的绝对路径

开启ssh，对外开放

打开ssh连接
sudo systemctl start sshd

// 停止ssh
sudo systemctl stop sshd

// 查看ssh状态
sudo systemctl status sshd

// 关闭开机启动
sudo systemctl disable sshd

常用命令

-- 问postgresql文件在哪个位置
whereis postgresql

-- 查看文件路径
sudo find / -name "postgresql.conf"

-- 查看端口是否存在
netstat -anp|grep 6789

- nc指令进行tcp通讯 
nc ip地址 

-- nc执行upd通讯  -4u  -6u
nc -4u ip地址

-- 根据关键字从历史命令查询
history | grep "ssh -fCNR"

-- 根据行号执行代码历史
-- 执行630行代码
!630

-- -f循环读取 -n显示文件的尾部行数  
tail -f -n 10 postgresql-11-main.log

-- 查看文件
more postgresql-11-main.log

vi文件命令

复制一行：光标移动到要复制的行上 --> 按“yy” --> 光标移动到要复制的位置 -->按“p”（“p”：将已复制的数据粘贴到光标的下一行，大写“P”：贴在光标的上一行）

删除一行：dd

撤销上一步操作：u

本地端口映射公网端口

ssh root@服务器地址
-- 进入后修改ssh配置文件，GetwayPorts yes
vi /etc/ssh/sshd_config
-- 重启ssh，是配置文件生效
systemctl restart sshd

--查看端口
netstat -anp|grep 6789

-- 查看当前firewall配置项
firewall-cmd --list-all

-- 给防火墙添加放行端口
firewall-cmd --permanent --add-port=6789/tcp

-- 重启防火墙
firewall-cmd --reload




ssh -fCNR 6789:127.0.0.1:6789 root@服务器地址
即可生效
